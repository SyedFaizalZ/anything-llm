## Workflow

1. Receive text input from user
2. Apply text processing operations (cleaning, tokenization, etc.)
3. Analyze text properties and generate statistics
4. Transform text as requested (case changes, pattern finding)
5. Present results with analysis

## Example Use Cases

- Count words, characters, sentences
- Remove duplicates and clean text
- Tokenize text into words
- Find word frequency and top N words
- Search for patterns or keywords
